== Description ==
This plugin adds WebsiteAlive Embedded Icon/Tracking Code in the footer section of your Wordpress site.

== Installation ==

1. Download wordpress-wsa.zip to your computer.
2. Create 'websitealive' folder in your '/wp-content/plugins/' directory (requires FTP access to your WordPress account).
3. Extract contents (3 files) from .zip file to 'websitealive' folder.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Configure plugin (go to the Settings/WebsiteAlive) in WordPress.
