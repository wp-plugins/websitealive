WebSitealive Wordpress Plugin

Requirements:

1) You will need to have a wordpress up and running.

How to install it:

1) Clone the repository and copy the files to the wp-content/plugins/ folder
2) Go to your wordpress to Plugins section, you will find websitealive plugin
3) Activate it and it will be ready to use.


Observation:

After you clone it, you will have the following structure folder on your wordpress:

wordpress/wp-content/plugins/websitealive-plugin
